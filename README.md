# Databricks Dashboard

This repository contains a Databricks project created to showcase my ability to work with data analysis, dashboard creation, and insight presentation using Databricks.

Contents

Dataset: The dataset used for this project is included in this repository.
Dashboard PDF: A PDF export of the Databricks dashboard.
Presentation: A presentation file is included where I explain the project in more detail, steps taken and key findings.

Project Overview

The goal of this project was to analyze the dataset, build a clear and interactive dashboard in Databricks, and present the most relevant insights in a professional format.

Purpose

This project is part of my portfolio and demonstrates my skills in:
- Data exploration and analysis
- Dashboard design in Databricks
- Communicating insights through presentations
